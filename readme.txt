详细操作说明：

详细操作演示可通过链接观看视频： https://easylink.cc/cl14cj 

1.添加图例：
	需要先点击场景树中的Trans节点，再点击二维绘图中的图形类型才可绘制图形
1.1添加直线
	鼠标左键点击两点即可生成直线段，一点为起点，另一点为终点
1.2添加闭合线
	鼠标左键点击多点，前后两点连线，最后鼠标右键点击为终点，生成闭合线

2.基本图形几何变换
2.1平移
	需要先点击Trans下的节点，再点击平移中的按钮实现相应操作，点击拖动时，鼠标按住左键不放并移动，即可对相应节点对应的图例进行拖动
2.2旋转
	需要先点击Trans下的节点，再点击旋转中的按钮实现相应操作，点击旋转时，鼠标按住右键不放并做圆周，即可对相应节点对应的图例进行旋转，
	同时若是点击shift+鼠标左键，可以自定义旋转的基准点，再进行旋转
2.3缩放
	需要先点击Trans下的节点，再点击缩放中的按钮实现相应操作，点击任意操作时，均可按住crtl+滚轮滚动进行相应的缩放，此时的缩放类型和按钮的
	缩放类型一致，同时若是点击shift+鼠标左键，可以自定义缩放的基准点，再进行缩放。
